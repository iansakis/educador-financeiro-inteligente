# 💰 Educador Financeiro Inteligente

Projeto desenvolvido como desafio da Digital Innovation One (DIO).

## Sobre o projeto

O objetivo é criar um prompt para uma Inteligência Artificial que funcione como um Educador Financeiro Inteligente, ajudando jovens adultos a organizar suas finanças e desenvolver melhores hábitos financeiros.

## Objetivos

- Organizar receitas e despesas.
- Criar metas financeiras.
- Aprender sobre investimentos.
- Melhorar o planejamento financeiro.

## Tecnologias utilizadas

- Markdown
- GitHub
- Engenharia de Prompt
